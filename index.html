Introduction

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, spacing, fonts, positioning, and responsive design.

CSS Syntax

A CSS rule consists of a selector and a declaration block.


selector {
    property: value;
}

Example:


p {
    color: blue;
    font-size: 18px;
}

Selectors

Selectors target HTML elements you want to style.

  • element – Targets all elements of that type
  • .class – Targets elements with a class
  • #id – Targets a specific element by ID

.class Selector

The .class selector targets elements with a specific class attribute. Classes can be reused on multiple elements. 


.highlight {
    color: yellow;
    font-weight: bold;
}

This text is highlighted.

a:hover Pseudo-Class

The :hover pseudo-class applies styles when a user moves the mouse over an element.


a:hover {
    color: orange;
    text-decoration: underline;
}

Typography & Line-Height

The line-height property controls vertical spacing between lines of text.


  p {
    font-size: 16px;
    line-height: 1.6;
}
